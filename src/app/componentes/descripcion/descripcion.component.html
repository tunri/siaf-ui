Descripción General

El presente sistema de diseño fue creado con finalidad de tener un conjunto de componentes reutilizables para acelerar el proceso de creación de distintas aplicaciones web, tanto para los futuros diseñadores y desarrolladores, para que pueden tener lista siempre una amplia colección de recursos, acompañados además de toda la información necesaria para trabajar con ellos adecuadamente.

Aplicativo base

Se ha creado un proyecto base, el cual contiene todos los componentes y estilos, para acelerar el proceso de desarrollo.
