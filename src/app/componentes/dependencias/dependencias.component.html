Bootstrap v. 5.2.0

El sistema de diseño está basado en el framework Bootstrap 5 ya que es una biblioteca de diseño robusta y multiplataforma. Para más información del framework puede visitar su documentación: https://getbootstrap.com/

Instalación

Instalar el paquete NPM

    npm install bootstrap@5.2.0
Importación

Para hacer uso de los estilos y componentes, basta con añadir el siguiente comando en su archivo styles.scss

    @import '~bootstrap/scss/bootstrap';
Perzonalicación

Para poder personalizar los estilos de Bootstrap 5, se deben agregar las siguientes reglas, recuerda que hay más variables que se pueden configurar, para eso puede visitar su documentación: https://getbootstrap.com/docs/5.2/customize/sass/ 

    $theme-colors: (
        "primary": $siaf-red,
        "secondary": $siaf-grey-10,
        "success":    $siaf-green,
        "info":       $siaf-blue,
        "warning":    $siaf-orange,
        "danger":     $siaf-red,
    );
Ng-bootstrap v. 13.0.0

Librería que nos proporciona algunos componentes para tan solo importarlos en nuestro proyecto y poder utilizarlos, estos que siempre nos traen problemas dado que requieren javascript. Documentación: https://ng-bootstrap.github.io

Instalación

Instalar el paquete NPM

    ng add @ng-bootstrap/ng-bootstrap
Importación

Importe el módulo NgbModule en su archivo app.module.ts
@ng-select v. 9.0.2

Sirve para acelerar el proceso de carga en las entradas y formularios. visitar documentación: https://github.com/ng-select

Instalación

Instalar el paquete NPM

    npm install @ng-select/ng-select@9.0.2
Importación

Importe el módulo NgSelectModule y el módulo FormsModule en su archivo app.module.ts
Personalización

Para poder utilizar los estilos basta con añadir el siguiente comando en su archivo styles.scss

    @import "~@ng-select/ng-select/themes/default.theme.css";
Nota Importante Recuerda, siempre es mejor tener actualizada a la última versión estable todas las dependencias
